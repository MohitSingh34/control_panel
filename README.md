# control_panel
This website is just a frontend way to control a ubuntu machine
Core things that can be controlled from the interface are:
1. Terminal having multiple sessions
2. screenshot
3. screenrecord
4. camera photo
5. camera video

I think that just by being able to control terminal, we can control everything on machine, no restrictions in terminal.
